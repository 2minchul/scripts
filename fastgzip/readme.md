# FastGZIP

## Usage

```
fastgzip [-options] [<input file>]

Usage of fastgzip:
    -d string
        output directory. Default is ./
    -f string
        input file. Required
    -n int
        split number. Saved as '{name}.{number}.gz' if 1 < n (default 1)
```

## Install

```shell
go install github.com/2minchul/scripts/fastgzip@latest
```
